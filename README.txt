Project Files
=============

"./README.txt"

"./Data": Folder for source files for the analysis

"./Data/UCI HAR Dataset": Downloaded from http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones

"./Scripts": Folder for scripts used to perform the analysis

"./Scripts/Download Project Files.R": The script to download the files from the source and unzip

"./Scripts/run_analysis.R": The script that performs the tasks set out in the project assignment

"./Output": Folder for results of the analysis

"./Output/SubjectActivities.csv": The tidy dataset output after the processing

"./Docs": Folder for documents explaining datasets
